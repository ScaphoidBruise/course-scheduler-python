About This Project

What is this?

UTPB Scheduler is a course scheduling tool built for students at The University of Texas Permian Basin. It lets you browse available sections, build a conflict-free weekly schedule, and explore the full course catalog.

Data Sources
  • Course catalog — scraped from the UTPB Smart Catalog API
  • Section schedules — scraped from the official UTPB course schedule pages (Spring, Summer, Fall 2026)
  • Term inference — derived from UTPB Falcon Maps degree-plan PDFs
  • Session dates — scraped from the UTPB academic calendar
How It Works

The data pipeline runs four Python scrapers that populate a single SQLite database. The web app reads from this database to display sections, detect time conflicts, and render the weekly calendar grid.

  1. scraper.py — fetches all courses from the UTPB catalog
  2. infer_term_from_degree_maps.py — infers typical term offerings from degree-plan PDFs
  3. scrape_sections.py — scrapes current semester section schedules
  4. scrape_session_dates.py — links session codes to calendar start/end dates
Built With

Python, Flask, SQLite, Bootstrap 5, vanilla JavaScript
