How to Use

1. Select a Term

On the Schedule page, use the Term dropdown at the top of the left panel to pick Spring, Summer, or Fall 2026. The section list updates to show only courses offered in that term.

2. Search and Filter

Use the search box to find courses by code or name. Narrow results further by subject, course level (1xxx–4xxx), or instruction mode. Click Filter to apply.

3. Add Sections

Click the + Add button next to any section. It will appear on the weekly calendar grid on the right. If the section conflicts with one you already added, the conflicting blocks will be highlighted in red.

4. Reading the Calendar

The grid shows Monday through Friday as columns and time of day on the vertical axis. Each section appears as an outlined block with a colored left border. The color is assigned per course so you can tell them apart at a glance.

Online and asynchronous sections don't have set meeting times, so they appear only in the Added Sections table below the grid.

5. Conflict Detection

If two sections share any day and their time ranges overlap, they are marked as conflicting. Conflict blocks on the grid get a red border, and the corresponding rows in the Added Sections table are highlighted. Online sections never conflict.

6. Remove or Clear

Remove individual sections using the × button in the Added Sections table. Use Clear All to start fresh.
